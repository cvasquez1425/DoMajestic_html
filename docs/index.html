Tools and Plugins

Build on top of Twitter Bootstrap

The Hairpress HTML template is build on top of SASS Twitter Boostrap HTML framework. All the components of the Bootstrap framework were left as they are during the development, so the template is upgrade-safe. The only file which has been changed and is a part of Bootstrap is _variables.scss, inside the ./lib/_variables.scss directory.

Here is the list of the variables which were changed:

  • $textColor
  • $baseFontSize
  • $navbarHeight
  • $navbarInverseBackground
  • $navbarInverseBackgroundHighlight

Find the documentation for Twitter Boostrap HERE.

You are allowed to use any component in your themes available in Twitter Bootstrap. However, for the performance reasons some of the components were commented out in the .lib/bootstrap.scss file. Uncomment the components you need in your project and recompile the project using $ compass compile when you cd to the root of the template in terminal.

Color Themes of the Template

Available themes

There are 5 different color themes available out of the box:

  • Violet (default)

  • Blue

  • Orange

  • Olive

  • Oil

How to change the theme

If you are familir with Compass/SASS, the easiest way to change the color theme is to change the variable $theme in the beginning of the main.scss file, which is located in ./lib/main.scss. Just set the variable to the name of the predefined themes (use lowercase), recompile the CSS files and you are all set:

$theme : "orange";

If you are unfamiliar with the Compass/SASS authoring framework, you can change the color theme by calling the other CSS file. The default theme is "violet". Just change the link to the other CSS file in head section of the HTML document from this:

<!--  = Custom styles =  -->
<link rel="stylesheet" href="stylesheets/main.css" />

to this:

<!--  = Custom styles =  -->
<link rel="stylesheet" href="stylesheets/orange.css" /><!--  or "blue.css" or "oil.css" or "olive.css"  -->

Contact and Appointment Form

Contact Form

The working contact form has been added to the Hairpress template in version v1.2.

The form is written in PHP and can be found in ./contact.php. Be sure that your server supports PHP and has all the required settings for build-in mail() PHP function.

All you need to do to enable the PHP form is to point the user to the contact.php instead of contact.html and change the following variable in the ./incl/send-email.php to your actual email address:

<?php

/* set the email of the recipient (your email) */
$recipient = "your.email@example.com";

The message will be sent directly from your website to the email address you have specified above and will have the following form:

----- Info about the sender -----

Name: [name]
Email: [email]
Phone: [phone]


----- Message -----

[the actual message from the user which sent the email]
